## Actores

- **Cliente:** usuario final que reserva y gestiona sus citas.
- **Barbero:** usuario interno que administra agenda, clientes, servicios y reservas.
- **Administrador:** rol con capacidad de administracion ampliada.
- **Google OAuth:** proveedor externo para autenticacion social.
