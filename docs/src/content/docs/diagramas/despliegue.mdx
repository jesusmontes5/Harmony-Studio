## Despliegue esperado

- Frontend principal: build estatico generado con Vite y desplegado en Vercel.
- Backend: jar de Spring Boot ejecutado en contenedor Docker.
- Proxy principal: Vercel reenvia `/api/*` hacia la VM de Azure.
- Proxy alternativo: Nginx reenvia `/api` al contenedor `service:8080`.
- Base de datos: Azure Database for MySQL Flexible Server con esquema `schema.sql`.
- Documentacion: sitio Starlight desplegado en Vercel como aplicacion estatica.
- Servicios externos: Google OAuth, SMTP opcional y Cloudinary para avatar.
