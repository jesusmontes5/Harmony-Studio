## Despliegue esperado

- Frontend: build estatico generado con Vite.
- Backend: jar de Spring Boot con variables de entorno.
- Base de datos: MySQL con esquema `schema.sql`.
- Documentacion: sitio Starlight desplegable como aplicacion estatica.
- Servicios externos: Google OAuth, SMTP opcional y Cloudinary para avatar.
