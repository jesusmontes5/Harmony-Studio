## Despliegue esperado

- Frontend: build estatico generado con Vite y servido desde Nginx.
- Backend: jar de Spring Boot ejecutado en contenedor Docker.
- Proxy: Nginx reenvia `/api` al contenedor `service:8080`.
- Base de datos: Azure Database for MySQL Flexible Server con esquema `schema.sql`.
- Documentacion: sitio Starlight desplegable como aplicacion estatica.
- Servicios externos: Google OAuth, SMTP opcional y Cloudinary para avatar.

Recursos actuales:

- Grupo de recursos: `rg-harmony-studio-es`.
- Region: `spaincentral`.
- VM: `vm-harmony-studio`.
- URL publica: `http://158.158.2.243/`.
- MySQL: `mysql-harmony-studio.mysql.database.azure.com`.
